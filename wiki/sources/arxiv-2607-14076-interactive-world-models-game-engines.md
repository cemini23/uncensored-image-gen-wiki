## Narrative

Survey framing interactive video world models through a classic **action → state → observation** game-engine loop. Four axes: player action control, game state dynamics, state–observation persistence, real-time interactive generation. Also announces a **Black Myth: Wukong** data engine (~90h gameplay with aligned actions, ground-truth states, visuals).
